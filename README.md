# Portfolio Website

This is a personal portfolio website built using **React.js**, showcasing projects, skills, resume, and a working contact form integrated with **EmailJS** for email notifications.

---

## Features

- **About Me Section** – Personal introduction and bio.
- **Services Section** – Highlighting skills like web development, mobile apps, and web design.
- **Projects/Portfolio Section** – Displays projects dynamically or via images.
- **Resume Section** – Option to show resume in PDF format.
- **Contact Form** – Fully functional contact form that sends messages directly to Gmail using EmailJS.
- **Testimonials Section** – Displays client feedback or references.
- Responsive design and modern UI with **Tailwind CSS** and **Material Icons**.

---

## Technologies Used

- **Frontend:** React.js, HTML5, CSS3, Tailwind CSS
- **State Management:** React Hooks
- **Email Integration:** EmailJS
- **Icons:** Ionicons
- **Other Tools:** Git, GitHub
